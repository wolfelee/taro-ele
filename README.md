# Taro3.0 仿饿了么 开发 h5 及微信小程序 跨端

> 编译预览:

```
npm run dev:h5
npm run dev:weapp
```

> 打包

```
npm run build:h5
npm run build:weapp
```

> 说明:
> 部分数据使用 ele 源数据,使用 eggjs 反向代理 [ele-serve](https://github.com/qingjuesama/taro-ele-serve) 采取 fs 本地保存 json
